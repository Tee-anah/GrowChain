# GrowChain - Gardening and Plant Cultivation System

A blockchain platform for tracking gardening activities, rewarding plant cultivation, and managing seed preservation with growth-based incentives.

## Features
- Gardening session tracking and rewards
- Gardener level progression system
- Seed preservation with time-based bonuses
- Specialty plant creation and cultivation
- Gardening workshop hosting

## Smart Contract Functions
- Track planting and cultivation sessions
- Manage seed preservation with growth rewards
- Create specialty plant varieties
- Host gardening workshops and classes
- Claim cultivation and growth rewards

## Usage
Promotes sustainable gardening through blockchain-verified cultivation tracking and horticultural achievement rewards.
